# TFP Program Analysis -- West Arsi (2019--2023)

## Overview

This repository contains analysis of Therapeutic Feeding Program (TFP)
data.

Analysis was performed entirely using ChatGPT's Python environment.

## Structure

-   data/ : cleaned dataset
-   analysis/ : Jupyter notebook
-   results/ : analysis outputs

## Key Files

-   tfp_cleaned_dataset.csv
-   yearly_admissions.csv

## Method

Data was:

1.  Loaded
2.  Cleaned
3.  Aggregated
4.  Analyzed

## Usage

Upload to GitHub and open the notebook for analysis.
